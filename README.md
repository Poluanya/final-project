# final-project

Этот проект создан для автоматического тестирования веб-сайта. Он помогает быстро проверять основные функции сайта, используя современные инструменты.

Основные возможности:

-Page Object Model: удобная структура для поддержки тестов.

-Генерация данных: используем Faker для создания уникальных тестовых данных.

-CI/CD: тесты автоматически запускаются при каждом обновлении кода на GitHub.

Как запустить проект:

1. Скачай проект и перейди в его папку.
2. Установи все нужные программы командой:

npm install

3. Запусти тесты командой:

npx playwright test

4. Посмотри отчет после запуска:

npx playwright show-report
